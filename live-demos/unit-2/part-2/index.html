Some brief information about this page

To help with accessibility, this column has a class of .text-white to change the text color based on the background.

They are using the Bootstrap grid, but we're not using Bootstrap styling.

You can verify this claim by checking the Bootstrap CSS file that's linked to this page!

Instead, we're using custom SCSS to generate some styling using the help of a color palette generator like Coolors.

Each of these columns has a class of .col-3 .py-xs and a background class like .bg-gold.
